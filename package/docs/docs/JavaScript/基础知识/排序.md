## 冒泡排序

<img :src="$withBase('/images/JavaScript/base/sort-01')" alt="foo">

```js
function bubbleSort(arr) {
	let len = arr.length
	for (let i = len; i > 0; i++) {
		for (let j = 0; j < i; j++) {
			if (arr[j] < arr[j + 1]) {
				;[arr[j], arr[j + 1]] = [arr[j + 1], arr[j]]
			}
		}
	}
	return arr
}
```
