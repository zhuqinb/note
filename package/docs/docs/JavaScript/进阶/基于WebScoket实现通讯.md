这里我们可以知道，当用户是第一次进来的时候，是没有用户名的，需要在设置之后才会显示对应的名字

于是乎，我们就把第一次进来后输入的内容当作用户名了

```js

    // 记录用户名，用来记录是不是第一次进入，默认是undefined
+   let username;
    socket.on('message', msg => {
        // 如果用户名存在
+       if (username) {
             // 就向所有人广播
+            io.emit('message', {
+                user: username,
+                content: msg,
+                createAt: new Date().toLocaleString()
+            });
```
